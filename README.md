# Crimson Vesper

A 2D metroidvania game — spiritual successor to Castlevania: Symphony of the Night.

Explore a sprawling, interconnected gothic world filled with secrets, cursed relics, and creatures of the night. Master a deep combat system, unlock new abilities to access previously unreachable areas, and uncover the mystery behind the Crimson Vesper.

## Project Structure

| Directory | Purpose |
|-----------|---------|
| `docs/` | Design documents — GDD, tech specs, art bible, audio direction, pipeline docs |
| `assets/` | All game assets: sprites, tilesets, backgrounds, UI, audio, fonts |
| `prototype/` | Early mockups, style explorations, animation tests |
| `godot/` | Godot engine project (initialized separately) |
| `tools/` | Custom scripts, asset pipeline utilities, generators |

## Engine

[Godot 4](https://godotengine.org/)
